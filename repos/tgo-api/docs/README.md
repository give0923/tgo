# TGO-Tech API Service

This directory contains all files related to the API Service (Core Business Logic).

## Files in this directory:
- `api_service_schema.sql` - Database schema for the API Service
- `TGO_API_Service_Documentation.yaml` - OpenAPI specification for API Service endpoints

## Service Overview:
- **Port**: 8080
- **Database**: `tgo_api_service`
- **Primary Focus**: User management, visitor interactions, business operations
- **Key Responsibilities**:
  - User authentication and authorization
  - Visitor management and tracking
  - Staff operations and assignments
  - Tagging and categorization system
  - Platform integrations
  - Project management (master copy)
